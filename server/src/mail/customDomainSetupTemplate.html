Please navigate to Kommunicate Dashboard and follow the instruction to set up a custom domain. The custom domain is required to deploy Helpcenter (knowledge-base) on your domain.
  1. Login to your domain administration panel and find DNS records management panel for your domain
  2. Then add new records
Type Name Value
CNAME helpcenter.<your-domain>.com helpcenter-proxy.kommunicate.io

Please verify that the Helpcenter is deployed on the specified domain after completing the configuration. Talk to us on Kommunicate support chat if you run into errors.

Additionally, if you have any problems with the setup of the domain, please contact your domain admin support team. They will be able to help you out.
